Gin-Gun (กินกัน)

แบ่งปันค่าข้าวกลุ่มเพื่อน จัดการยอดค้างชำระ แชทบอกกล่าว และกำหนดการจ่ายเงิน

ผู้ใช้: ...

เชื่อมต่อฐานข้อมูล Firebase

* วางการตั้งค่า Firebase Web App ในรูปแบบ JSON (สามารถคัดลอกได้จากหน้าตั้งค่าโครงการใน Firebase Console)

* ลบข้อความในช่องนี้ทั้งหมดและบันทึกหากต้องการสลับกลับไปใช้งานโหมดจำลองออฟไลน์ (Offline/Mock Mode)

ตั้งค่าระบบ LINE

* รับ Token ได้จาก LINE Developers Console (Messaging API Channel) ข้อความจะส่งแบบ Broadcast ไปหาผู้ใช้ทุกคนที่ติดตามไลน์บอทของคุณ

* URL สำหรับรับสิทธิ์ Login จาก LINE (เช่น URL ของ Firebase Functions หรือ Apps Script Callback Proxy)

คำแนะนำการเชื่อมต่อ Firebase:

  1. สร้างโปรเจกต์ใหม่ใน Firebase Console
  2. เปิดใช้งาน Cloud Firestore และ Firebase Storage ในโหมดทดสอบ (Test Mode) หรือกำหนด Rules ให้อ่าน/เขียนได้
  3. สร้างเว็บแอปพลิเคชัน (Web App) ในโปรเจกต์ และคัดลอกอ็อบเจกต์ firebaseConfig มาแปลงเป็นรูปแบบ JSON วางลงในช่องด้านบน
  4. กดบันทึกการตั้งค่า ระบบจะเชื่อมต่อและซิงก์ข้อมูลกับ Firebase ทันที!
ขณะนี้แอปทำงานใน โหมดจำลองในเบราว์เซอร์ (Offline Demo) ยินดีต้อนรับการทดลองใช้งาน
