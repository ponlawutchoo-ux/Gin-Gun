# Gin-Gun (กินกัน) - Single Page Web Application 💸

แอปพลิเคชันหน้าเดียว (Single Page Application) สำหรับบันทึกและคำนวณการหารค่าอาหารในกลุ่มเพื่อน 3 คน (นาย A, นาย B, นาย C) พัฒนาโดยใช้ HTML, Tailwind CSS v3 และ Vanilla JavaScript เชื่อมต่อระบบหลังบ้านผ่าน Firebase (Firestore & Storage)

---

## 📁 โครงสร้างโปรเจกต์ (Project Structure)

- **[index.html](file:///c:/Users/user/Downloads/1/1/index.html)** - หน้าเว็บแอปพลิเคชันหลัก (SPA) มีระบบกรอกฟอร์มบันทึกบิล แสดงผล Net Balance และระบบเคลียร์บิล (Settlement Suggestions)
- **[readme/README.md](file:///c:/Users/user/Downloads/1/1/readme/README.md)** - คู่มือแนะนำขั้นตอนการสร้างและตั้งค่าโครงการ Firebase แบบละเอียด

---

## ✨ ฟีเจอร์เด่น (Key Features)

1. **โหมดทดลองใช้งาน (Demo/Mock Mode)**
   - หากผู้ใช้ยังไม่ได้กำหนดค่า URL ระบบจะทำงานแบบออฟไลน์จำลองการทำงานทันที ทำให้เห็นหน้าตาและการคำนวณบิลได้โดยไม่ต้องติดตั้งสเปรดชีตก่อน
2. **ระบบ Net Balance สะท้อนสถานะจริง**
   - คำนวณยอดเงินสะสมของเพื่อนแต่ละคนในกลุ่มเป็นบวก (รอรับเงิน) หรือลบ (ต้องโอน)
3. **คำแนะนำการเคลียร์บิล (Settle Bills Generator)**
   - วิเคราะห์หนี้และบอกขั้นตอนทันทีว่า **ใครควรโอนให้ใคร เท่าไหร่** เพื่อยุติยอดคงค้างให้ได้เร็วที่สุดด้วยจำนวนครั้งการโอนที่น้อยที่สุด (Greedy Settlement Algorithm)
4. **ดีไซน์สวยงาม ทันสมัย (Premium Minimal CSS)**
   - เลือกใช้คู่สีเขียวธรรมชาติ สบายตา อารมณ์โปร่งใสเรื่องเงินๆ ทองๆ พร้อมเอฟเฟกต์ Glassmorphism สวยงาม คลีน และรองรับการทำงานในหน้าจอมือถือ (Responsive Design)

## 🚀 ขั้นตอนการเริ่มใช้งานด่วน

1. ดับเบิลคลิกเปิดไฟล์ `index.html` บนเบราว์เซอร์ของคุณเพื่อเปิดหน้าเว็บ
2. ทำตามขั้นตอนการเชื่อมต่อฐานข้อมูล Firebase ในคู่มือ [readme/README.md](file:///c:/Users/user/Downloads/1/1/readme/README.md)
3. ใส่ Firebase Config JSON ในหน้าจอตั้งค่า แล้วเริ่มต้นบันทึกข้อมูลจริงได้เลย!
